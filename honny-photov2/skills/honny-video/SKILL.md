# HonnyVideo Skill

使用 RunningHub Wan2.2 API 进行图生视频生成。

## 功能

| 功能 | 说明 |
|------|------|
| **图生视频** | 上传图片生成动态视频 |
| **队列重试** | 并发满时自动等待重试 |
| **提示词优化** | 支持动作+表情+环境+运镜 |

## 配置

### 环境变量

在 `~/.openclaw/skills/honny-video/.env` 中配置：

```bash
# RunningHub API Key
RUNNINGHUB_API_KEY=你的API密钥

# 工作流 ID（图生视频-强一致性）
WORKFLOW_ID=2016727774387511298
```

### 获取 API Key

1. 访问 https://www.runninghub.cn
2. 登录 → 个人中心 → API
3. 创建 API Key

## 使用方法

### 提示词公式

```
(主体描述) + (表情动作) + (环境氛围) + (运镜方式)
```

**示例**：
- "一个女孩在玩耍"
- "看着镜头，比耶的手势，微笑，下雨天，镜头拉远"
- "可爱的小猫，伸懒腰，阳光明媚，镜头推进"

### 基本命令

```
生成一段视频
做一段图生视频
```

## 输出格式

```
🎬 视频生成成功！

📝 提示词：{使用的提示词}
🎥 视频：{视频URL}

⏱️ 耗时：{时间}秒
💰 消耗：{金币}金币
```

## 注意事项

- 需要配置 RUNNINGHUB_API_KEY
- 需要提供图片（作为参考图）
- 生成可能需要等待队列
- 视频 URL 有效期较短，请及时保存

---

*基于 RunningHub API v2*
